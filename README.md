Projet : Recherche du chemin optimal entre ARAD et BUCHAREST
Description
Ce document présente une analyse comparative de plusieurs itinéraires entre les villes d'ARAD et de BUCHAREST. L'objectif est d'identifier le chemin le plus optimal en termes de distance parcourue (en kilomètres).

Itinéraires analysés
Le document examine 6 itinéraires différents :

ARAD → SIBIU → FAGARAS → BUCHAREST
Distance : 450 km

ARAD → SIBIU → RIMMICU VILCEA → CRAIOVA → PITESTI → BUCHAREST
Distance : 605 km

ARAD → SIBIU → RIMMICU VILCEA → PITESTI → BUCHAREST
Distance : 418 km

ARAD → TIMISORA → LUGOJ → MEHADIA → DOBRETA → CRAIOVA → RIMMICU VILCEA → PITESTI → BUCHAREST
Distance : 838 km

ARAD → TIMISORA → LUGOJ → MEHADIA → DOBRETA → CRAIOVA → PITESTI → BUCHAREST
Distance : 733 km

ARAD → ZERIND → ORADEA → SIBIU → FAGARAS → BUCHAREST
Distance : 607 km

Solution
L'itinéraire le plus optimal est :
ARAD → SIBIU → RIMMICU VILCEA → PITESTI → BUCHAREST
Avec une distance totale de 418 km.

Notes
L'optimalité est évaluée uniquement sur la base de la distance totale du trajet.

Toutes les distances sont exprimées en kilomètres (km).
